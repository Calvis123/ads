# Barak Pathways Landing Page

React + Tailwind CSS landing page for Barak Pathways (study abroad agency).

## Run locally

```powershell
npm.cmd install
npm.cmd run dev -- --host 127.0.0.1 --port 8091
```

Open:

`http://127.0.0.1:8091/`

## Build

```powershell
npm.cmd run build
npm.cmd run preview -- --host 127.0.0.1 --port 8091
```
